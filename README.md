# 🎮 Track Life Like a Game — Personal Habit Tracker

A gamified daily habit tracker built for personal use — because generic habit apps felt boring, so I built one with streaks, stats, and 16 fully custom visual themes (including anime-inspired ones).

> Built with plain HTML, CSS, and JavaScript — no frameworks, runs entirely in the browser.

---

## ✨ Features

- **Daily habit check-ins** with an interactive calendar grid to log progress day by day
- **Streak tracking** — current streak, best streak, and total days completed per habit
- **Top Habits leaderboard** — see which habits you're most consistent with at a glance
- **Stat cards** — quick-glance dashboard of your overall progress
- **16 visual themes**, including:
  - High-energy packs: *Adrenaline, Midnight, Neon Overdrive, Blood Moon, Shadow Ops*
  - Soft/calm packs: *Bloom, Ocean, Peach Sunset, Lavender Mist, Golden Glow*
  - Neutral/focus packs: *Focus, Slate*
  - Anime-inspired packs: *Shadow Monarch (Solo Leveling), Nindo Way (Naruto), Grand Line (One Piece), Hinokami (Demon Slayer)* — each with its own color palette and motif pattern
- **Custom typography & text effects** — multiple display fonts and effects (gradient glow, neon pulse, shimmer, fire, ice, metallic, glitch) for personalizing your name/title
- **Custom background image support** for any theme
- **Multi-user support** — switch between different named profiles, each with its own isolated habit data
- **Export / Import backup** — save your data as a JSON file and restore it anytime
- **Add / delete habits** with confirmation prompts to avoid accidental data loss
- Fully **responsive**, self-contained in a single HTML file — no installation, no build step

## 🚀 Getting Started

No installation needed — it's a single self-contained HTML file.

1. Clone or download this repository
2. Open `My_Tracker.html` directly in any modern browser
3. Add your habits and start tracking!

Or try it live: **[Live Demo](#)** *(add your GitHub Pages link here once enabled)*

### Enabling GitHub Pages (to get a live demo link)
1. Go to your repo → **Settings → Pages**
2. Under "Branch," select `main` and `/ (root)`
3. Save — GitHub will give you a live URL in a minute or two

## 🛠️ Tech Stack

- **HTML5** — structure
- **CSS3** — theming system, animations, responsive layout
- **Vanilla JavaScript** — state management, habit logic, streak calculations, import/export
- **Browser localStorage** — data persistence (no backend required)

---

## 💡 Why I Built This

I wanted a habit tracker that actually made me *want* to open it every day, instead of feeling like another chore app. So instead of using an existing tool, I designed and built my own — with a theming system flexible enough to match my mood, and just enough "game-like" feedback (streaks, stats, leaderboards) to make consistency feel rewarding.

This project was a deep dive into CSS theming at scale, state management in vanilla JS, and building something end-to-end for a real, personal use case rather than a course assignment.
